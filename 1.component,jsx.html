What is component?

Component is a building block of website. এইটা কোন website এর ছোট অংশ।

Structure of component is following:

export default function Profile(){
return
}

Note: এখানে component এর ক্ষেত্রে function এর নাম এর first letter টা capital হবে। e.g Profile. একটা component এর ভিতরে আলাদা কোন component use করা হয় না। আর যদি করা হয় তাহলে performance issue হয়।

Advantage

  1. Code reusability: বার বার usr করতে পারা।
  2. Fast development

What is JSX?

JSX stands for JavaScript XML = JSX

How JSX Works?

JSX File -> JSX Compiler (like Babel) -> converts to JavaScript file -> create a Virtual DOM using JS file -> then compare if any changes occures

Rules of JSX

  • It returns a single element like function. ণিচে একটা single div return করতেসে। i.e img

    • export default function Profile(){
    return
    div img src="" alt="" srcset="" />
    } div

  • Or you can return multiple element. নিম্নে multiple element return করতেসে i.e h1 & img একটা fragment or self closing tag এর মধ্যে <> ।

    • export default function Profile(){ return <>
    h1 Hello h1
    img src="" alt="" srcset="" /> img
    }

  • Template string এর পরিবর্তে এখানে curly braces use করতে হয়।
